# php-simple-calculator
A beginner-friendly **PHP calculator web app** built using PHP, HTML, and CSS
This project performs basic arithmethic operations and validates user input.

FEATURES
+ Addition
- Subtraction
* Multiplication
/ Division
Input validation
Error handling (e.g division by zero)
Simple and clean UI
